JVM运行时动画解析 - 002

i = i++ 字节码执行过程可视化演示

Java源代码

public class Test {
    public static void main(String[] args) {
        int i = 0;
        i = i++; // 为什么 i 最终还是 0？
        System.out.println(i); // 输出: 0
    }
}

字节码指令

PC 程序计数器
-
等待执行
  • 0 iconst_0 将常量0压入栈
  • 1 istore_1 栈顶值存入局部变量1(i)
  • 2 iload_1 局部变量1(i)加载到栈
  • 3 iinc 1 by 1 局部变量1(i)自增1
  • 6 istore_1 栈顶值存入局部变量1(i)
