本文详细列举了IntelliJ IDEA中Mac与Windows系统下的快捷键对照表，涵盖按键含义（如⌘、⌥、⌃等）、代码生成、追踪、高效编辑、重构、工具类操作及常规文件操作等实用快捷键，帮助开发者快速提升编码效率。

<!-- more -->

# 按键含义
+ `⌘ Command`键相当于Windows中的Control键（Ctrl）
+ `⌥ Option`键相当于Windows系统中的Alt键
+ `⌃ Control`它本身没有实质性的功能，但和其他键组合起来可以完成很多操作
+ `⇧ shift`
+ `⬆ 上箭头`
+ `⬇ 下箭头`

# 代码生成快捷键
| **Windows** | **Mac** | **描述** |
| --- | --- | --- |
| `Alt+Insert` | `⌘+N` | 光标在代码编辑界面时生成构造函数、Get、Set、和 Object 类方法重载<br/>光标在左侧工程结构时创建新类/包等 |
| `Ctrl+Alt+T` | `⌘+⌥+T` | 可以把代码包在一个块内，例如：try/catch,if,if else,try catch,do while |
| `Ctrl+O` | `⌃+O` | 重写父类方法 |
| `Ctrl+I` | `⌃+I` | 实现父类方法 |
| | `⌃+回车` | 最全的一个按键 |
| `Ctrl+Alt+M` | `⌘+⌥+M` | 将当前选中到代码块抽取为方法 |
| `PSVM+回车` | `PSVM+回车` | 自动生成main函数 |
| `SOUT+回车` | `SOUT+回车` | 自动生成System.out.println() |
| `FORI+回车` | `FORI+回车` | 生成For循环模板 |
| `ITER+回车` | `ITER+回车` | 生成ForEach循环 |

| **Windows** | **Mac** | **描述** |
| --- | --- | --- |

| `Ctrl+Alt+鼠标点击` | `⌘+⌥+鼠标点击` | 查看实现类源码 |
| `Ctrl+N` | `⌘+O` | 查找并打开类文件 |
| | `⌘+G` | 查找下一个 |
| `Double Shift` | `Double Shift` | 快速查找 |
| `Ctrl+H` | `⌃+H` | 查看类的继承体系 |
| `Ctrl+B` | `⌘+B` | 查看源码 |
| `Ctrl+Alt+B` | `⌘+⌥+B` | 查看实现类源码 |
| `Ctrl+Alt+U` | `⌘+⌥+U` | 查看当前类图 |
| | `⌘+⌥+7` | 展示类中所有方法 |
| `Ctrl+F12` | `⌘+F12` | 查找当前类的方法 |


# 高效编辑快捷键
| **Windows** | **Mac** | **描述** |
| --- | --- | --- |
| | `⌘+左右方向键` | 将光标移动到行头或行尾 |
| | `Shift+⌥+上下键` | 上下移动代码行 |
| | `⌘+Shift+U` | 大小写转换 |
| | `⌘+Shift+8` | 开启列编辑 |
| | `⌘+Z` | 撤销改动 |
| | `⌘+W` | 关闭当前类窗口 |
| | `⌘+S` | 保存文件 |
| | `⌘+Delete` | 删除当前行 |
| | `⌘+D` | 复制上一行，如果有光标的话，连着复制光标选择内容，不自动换行 |
| | `⌘+E ` | 最近浏览的文体 |
| | `⌘+⌥+L` | 格式化代码 |
